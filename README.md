# Emotions Detection Using Game AI

## Introduction
The Emotions Detection Using Game AI project aims to create an engaging game that leverages artificial intelligence to detect and interpret human emotions. The game provides an interactive environment where players can experience emotion recognition in real time and understand how AI can be applied in a gaming context.

## Features
- Real-time emotion detection
- Interactive gameplay with AI feedback
- User-friendly interface
- Multi-platform compatibility
- Leaderboard for tracking scores

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/SIDDMATRIX/Emotions-detection-using-game-ai-.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Emotions-detection-using-game-ai-
   ```
3. Install required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## How to Play
1. Launch the game from your terminal or command prompt.
2. Follow the on-screen instructions to begin.
3. Utilize your webcam to allow the game to detect your emotions.
4. Engage with the challenges presented by the game, aiming to achieve the highest score possible!
